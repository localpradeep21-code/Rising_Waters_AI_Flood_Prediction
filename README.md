# 🌊 Rising Waters – AI-Powered Flood Prediction System

## Project Overview

Rising Waters is an AI-powered Flood Prediction System developed using Machine Learning and Flask. The application predicts the likelihood of floods based on weather-related parameters such as rainfall, cloud percentage, and season. It helps authorities and disaster management teams take preventive actions by providing early flood risk predictions.

------------------------------------------------------------------

## Objectives

* Predict flood risk using Machine Learning.
* Provide early warnings for flood-prone areas.
* Support disaster management and resource allocation.
* Improve decision-making using data-driven predictions.

---------------------------------------------------------------

## Features

* AI-based Flood Prediction
* User-Friendly Web Interface
* Fast Prediction Results
* High Accuracy Model (XGBoost)

## Run the Project

Train the model:

```bash

```

Run the Flask application:

```bash
python app.py
```

Open your browser:

